# xUSSR+ Ivolga Addon
xUSSR+ Ivolga Addon is a NewGRF for OpenTTD, bringing EG2Tv "Ivolga" train family to the game.<br>

## Features<br>
* Graphics randomization for more variety
* Real world train specs
* Fully customizable costs and capacity
